serverguard.command.Run(player, command, bIsSilent, ...)
Makes a player run a command.

Parameters:

  • player player The player running the command. You should not pass this argument clientside.
  • command string The unique ID of the command to run.
  • bIsSilent boolean Whether or not to display a notification.
  • ... Any arguments to pass to the command.

Returns:

    bool Whether or not the command exists.
