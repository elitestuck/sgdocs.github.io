GetCurrentVersion()
Returns the version of this server's copy of ServerGuard.

Returns:

    string Version number with format #.#.#.
GetLatestVersion()
Returns the latest version of ServerGuard. This could be incorrect, as the server has to wait for the HTTP request to complete.

Returns:

    string Version number with format #.#.#.
PrintConsole(text)
Prints to console with the time and a [ServerGuard] prefix.

Parameters:

  • text string The text to print to the console.
ParseFile(filePath, state)
Includes files based on their state (client, server, shared).

Parameters:

  • filePath string The file to parse.
  • state SERVERGUARD.STATE State enumeration to specify how to parse it.

Usage:

    serverguard.ParseFile("tools/sv_administration.lua", SERVERGUARD.STATE.SERVER);
Notify(player, ...)
Sends a message in a player's chat box. Set player to nil or LocalPlayer() when using clientside.

Parameters:

  • player player The player to send the notification to. Specify nil to send to everyone.
  • ... Any notify enumeration followed by text.

Usage:

    serverguard.Notify(nil, SERVERGUARD.NOTIFY.GREEN, "This", SERVERGUARD.NOTIFY.WHITE, " is a test.");

SERVERGUARD.STATE
Enumeration table to specify file state.

Fields:

  • SERVER Server state.
  • CLIENT Client state.
  • SHARED Server and client state.

See also:

    serverguard.ParseFile
SERVERGUARD.IMMUNITY
Enumeration table to specify immunity comparisons.

Fields:

  • EQUAL Immunities are equal.
  • LESS Target immunity is less than the given one.
  • LESSOREQUAL Target immunity is less than or equal to the given one.
  • ANY Any immunity works.
