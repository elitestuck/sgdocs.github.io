serverguard.player:SetRank(player, rank[, bSaveless])
Sets a player's rank.

  • player player The player to set the rank of.
  • rank string The unique ID of the rank to set as.
  • bSaveless boolean Whether or not to save the player's rank. Defaults to false; (optional)

Returns:

    bool Whether or not the operation was successful.
