ExecuteOnPlayers(identifier, callingPlayer, requiredImmunity, func)
Attempts to find all connected players matching the given query, and runs a function with that player as an argument. Valid identifiers include name, Steam ID, 64-bit Steam ID, IP address, a comma-separated list, or a special tag. Tags include ^ for self, * for all, and #name for all players with the given rank. You can use ! to negate the action.

Parameters:

  • identifier string String to try and match players with.
  • callingPlayer player Player who initiated the query.
  • requiredImmunity number The required immunity level between the calling player and any targets for the function to run.
  • func function Function to run with all applicable targets.

Returns:

    table List of applicable players.
FindPlayer(identifier, user, bNoMessage)
Attempts to find a connected player by a given identifier. Valid identifiers include name, Steam ID, 64-bit Steam ID, and IP address.

Parameters:

  • bNoMessage boolean Whether or not to provide feedback to the player.

Converts an ISO 8601 duration timestamp used by YouTube to seconds. Note: this is not entirely accurate to the ISO standard - it's only tested for use with YouTube's duration format.

ParseDuration(input)
Converts a duration (e.g 1d12h) to minutes. Clamps all numbers between 0 and 99.

Parameters:

  • input string The input duration. This can be a number, but it will return that number.

Returns:

  1. number The duration in minutes.
  2. text The duration in text form (1 year, 3 days, etc).
  3. bool Whether or not any input values have been clamped.

Obtains a list of elements suitable for serverguard.Notify for the given player list. It is assumed that there is at least one target.

Fields:
